# EDEN-Graph: Network Analysis & Market Clustering Module

## Project Overview
This Open Science Framework (OSF) project serves as the immutable data and code repository for the software **"EDEN-Graph"**. It includes the reference implementation for the paper **"EDEN: Towards a Computational Framework to Align Incentives in Healthy Aging"** (BIOSTEC 2025) and extends it with a new **Market Clustering** module. 

The goal of this project is to ensure the **transparency**, **reproducibility**, and **long-term preservation** of the computational methods used to map incentive alignment in digital health ecosystems.

## Data Management & Reproducibility
The project adheres to **FAIR Principles** (Findable, Accessible, Interoperable, Reusable).

### 1. "Frozen" Replication Layer
This repository captures the specific version of the codebase (`v1.0-academic`) and dataset used to generate the results discussed in the publication. While the active software may evolve on GitHub, this snapshot guarantees that the scientific claims made in 2025 remain verifiable.

### 2. Artifacts
*   **Computational Core (`app.py`, `market_analyzer.py`)**: The exact Python implementation of the NMF and Hierarchical Clustering algorithms.
*   **Primary Dataset (`0_Longevity World.xlsx`)**: The structured corpus of 128 stakeholder value propositions.
*   **Market Data (`DHT_T2D...xlsx`)**: The supporting financial metrics used for the market segmentation analysis.
